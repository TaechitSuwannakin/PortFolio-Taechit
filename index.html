เกี่ยวกับฉัน

สวัสดีครับ ผมเป็นนักศึกษาชั้นปีที่ 3 สาขาคณิตศาสตร์เชิงคอมพิวเตอร์ มหาวิทยาลัยเทคโนโลยีพระจอมเกล้าพระนครเหนือ

โดยผมมีความสนใจอย่างมากในการพัฒนาเว็บไซต์ (Web Development) ในฝั่งของ Front-end
และอีกทั้งยังมีความชอบในส่วนของการออกแบบ UX / UI Design เพราะผมเชื่อว่าโปรแกรมที่ดีต้องมาพร้อมกับการใช้งานที่ง่ายและสวยงาม ผมจึงให้ความสำคัญกับ การออกแบบประสบการณ์ผู้ใช้งาน (UX/UI Design) ควบคู่ไปด้วย เพื่อสร้างสรรค์หน้าเว็บที่ไม่ใช่แค่ทำงานได้ แต่ต้องสร้างความประทับใจให้กับผู้ใช้งานจริง และในช่วงเวลานี้ผมกำลังมองหาโอกาสในการฝึกงานสหกิจศึกษา (Cooperative Education) เพื่อนำความรู้ที่มีไปประยุกต์ใช้และเก็บเกี่ยวประสบการณ์การทำงานจริง เพื่อพัฒนาสู่การเป็น Web Developer คุณภาพในอนาคตครับครับ
